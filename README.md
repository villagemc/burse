# Установление и запуск проекта:

1. npm install
2. npm run dev

## Билд проекта и его запуск:

1. npm run build
2. npm run preview

### Информация о проекте:

Проект посвящен биржам в Telegram Web App.

### Технологии:

- Vite
- HTML
- CSS Nesting
- AutoPrefixer
- TypeScript
- React
- SVGR
- GitHub Pages
- TanStack Router (File Path) + Devtools
